# Wisecow-app

Installations:
1. Docker
2. Kubernetes
3. Fortune-mod Cowsay
4. Git

### All the commands for the installations can be accessed through Internet or also checkout 'https://github.com/LondheShubham153/kubestarter.git' for kubernetes installation.
